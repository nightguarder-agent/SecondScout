# SecondScout Roadmap

## Current Status (v0.5)
- [x] Multi-provider search for Czech Republic (Bazos, Sbazar, Cyklobazar, Aukro).
- [x] Basic noise filtering (accessories, wanted ads).
- [x] Anti-scam integration (flagging known scammers).
- [x] Basic Watcher system (manual run).
- [x] Image proxying to bypass hotlink protection.

## Short-term (v0.6) - *In Progress*
- [ ] **Advanced Keyword Selection**:
    - Support for negative keywords (e.g., `iphone -pro -max`).
    - Multi-word AND logic improvement.
    - Automatic category detection based on keywords.
- [ ] **Kleinanzeigen.de Support**:
    - Full scraper implementation for Germany's largest classifieds.
    - Regional filtering for DE.

## Mid-term (v0.7)
- [ ] **Notification System**:
    - Email alerts for Watchers.
    - Discord/Telegram webhook integration.
- [ ] **User Authentication**:
    - Personal watcher lists.
    - History of viewed items.
- [ ] **Price Tracking**:
    - Monitor price drops for watched items.

## Long-term (v1.0)
- [ ] **Mobile App**:
    - React Native or PWA with push notifications.
- [ ] **AI-powered Filtering**:
    - Use LLMs to better classify items and filter out noise with high precision.
    - Automatic price evaluation (comparing with market average).
- [ ] **Support for more regions**:
    - Poland (Allegro/OLX).
    - Austria (Willhaben).
